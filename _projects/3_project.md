**RA-HTGNN** — a Relation-Attention Temporal Heterogeneous Graph Neural Network for predicting **credit-risk contagion** across firms. Built on a dynamic heterogeneous corporate network with seven relation types over 3,797 Chinese A-share companies (2010–2024), it combines per-relation graph convolution, relation-level attention fusion, and GRU-based temporal encoding. A stacking ensemble with Random Forest reaches **AUC = 0.839**.
