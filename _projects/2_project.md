An **information-theoretic** account of *when synthetic data helps* large language model training. Synthetic data improves a model only when the generation–training loop is **information-open** — shaped by external signals such as verifiers, environments, or rubrics. When the loop is **information-closed**, the data processing inequality predicts that task-relevant information can only decrease, making model collapse an expected outcome.
